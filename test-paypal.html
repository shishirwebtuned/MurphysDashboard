🔍 PayPal Connection Tester

This tool tests if your PayPal sandbox credentials are working correctly.

📝 If Test Fails - Get Fresh Credentials:

  1. Go to PayPal Developer Dashboard
  2. Login with your PayPal account
  3. Go to Apps & Credentials
  4. Under "Sandbox", click "Create App"
  5. Name it "Murphy's Technology" and click Create
  6. Copy the Client ID and Secret
  7. Update your .env file with these new credentials

👤 Create Test Buyer Account:

  1. In PayPal Developer Dashboard, go to Testing Tools → Sandbox Accounts
  2. Click "Create Account"
  3. Select "Personal" account type
  4. Country: United States
  5. Add Balance: $500
  6. Click Create
  7. Use this account's email/password to test payments
